== logstash-filter-yara

=== Example filter plugin

[,logstash]
----
filter{
  yara {
    yara_rule => '/opt/rb/var/yara_rules/rules.yara'
    yara_path => '/opt/rb/bin/yara'
    file => '[path]'
    score_name => 'fb_yara'
    weight => 1.0
    target => 'yara'
    latency_name => 'yara_latency'
  }
}
----

=== Description

This is an example of Logstash filter plugin for Yara.

[id="plugins-{type}s-{plugin}-yara_rule"]
===== `yara_rule`

* Value type is `String`.
* Default value is `"/opt/rb/var/yara_rules/rules.yara"`

Path where yara rules is located.

[id="plugins-{type}s-{plugin}-yara_path"]
===== `yara_path`

* Value type is `String`.
* Default value is `"/opt/rb/bin/yara"`

Path where Yara binary is located.

[id="plugins-{type}s-{plugin}-file"]
===== `file`

* Value type is `String`.
* Default value is `"[path]"`

Logstash event field (key) where the file to be analyzed is located.

[id="plugins-{type}s-{plugin}-score_name"]
===== `score_name`

* Value type is `String`.
* Default value is `"fb_yara"`

Logstash event field (key) where the score is going to be located.

[id="plugins-{type}s-{plugin}-weight"]
===== `weight`

* Value type is `Float`.
* Default value is `1.0`

Malware score is 100 if Yara tag the file as a virus. If not, score is 0.

[id="plugins-{type}s-{plugin}-latency_name"]
===== `latency_name`

* Value type is `String`.
* Default value is `"yara_latency"`

Logstash event field (key) where the latency is going to be located.

[id="plugins-{type}s-{plugin}-target"]
===== `target`

* Value type is `String`.
* Default value is `"yara"`

Logstash event field (key) where the JSON is going to be located.
